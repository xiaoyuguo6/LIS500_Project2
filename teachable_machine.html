Project #3: Teachable Machine

Can you teach a machine? We’ve learned about machine learning, and now we will put it into practice through this tutorial and assignment. Teachable Machines, developed by Google, is designed to help people learn the basics of artificial intelligence or big data perception. You only need HTML/CSS knowledge to create a webpage, along with a bit of JavaScript included in the tutorial. A computer camera will also be helpful. Follow the tutorial to get started.

Group Work Instructions

You must complete this work in groups of 2 or 3. This is not a solo project.

  • First Iteration Deadline: December 3
  • Feedback Deadline: December 6
  • Final Iteration Deadline: December 10

Tasks

  • Create a new page on your project website to upload your own version of a voice or image classification algorithm using Teachable Machines.
  • Write a project description about creating the algorithm (1000+ words).
  • Reflect on lessons from “Unmasking AI” that resonated with you after completing this assignment.
  • Embed a video showing how your algorithm works.
  • Attach as much of your ML application/experiment as possible to the website (e.g., separate pages).
  • Upload your code to GitHub and include a repository link on the webpage.
  • Post your link in the discussion section below.
